# ArcWave

ArcWave is an audio player Android application built with Kotlin, specifically designed for users who handle large audio files.

## Features

*   **Circular Seekbar**: A unique circular seekbar allows for precise position adjustments within a 5-minute range, making it easy to navigate long audio tracks.

## Modules

*   `kelly3d`: This is the main application module.

## Building and Running

To build and run the application, you can use the `androidApp` run configuration in Android Studio or execute the following command in your terminal:

```shell
./gradlew :kelly3d:assembleDebug
```
